Welcome!

In this presentation, we’ll start with some basic unit tests and progress to more advanced integration testing. We’ll generate code coverage reports that show exactly which lines of code have been tested. We’ll create some integration tests using a GUI that’s as simple as recording an MS Word macro. Finally, we’ll set up a fully automated test harness using a headless browser that can automatically test any PHP code, no matter how it was written.
