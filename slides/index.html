Installing The Tools

Things we will need

  • Some unit tests
  • Tool to run the tests
  • Something to control the browser

These two are required

  • PHPUnit 4.8
  • Selenium Stand-Alone Server 2.47.0

These three are optional

  • PhantomJS 2.0 (Patched)
  • Selenium IDE 2.9.0
  • Selenium 2 PHP Formatter                  

Required Prerequisites

(You probably already have them installed)

  • PHP 5.3.3 (or higher)
  • Web server w/ PHP Configured
  • Java Runtime
  • Firefox
  • Xdebug Extension (for code coverage)
