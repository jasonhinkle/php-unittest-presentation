What is Unit Testing?

  • Automated testing of your application
  • Shows you how much of your code is tested

Why Write Unit Tests?

  • Helps you find bugs
  • Does the tedious testing for you
  • Saves you time during development
  • Lets you know if you broke something
  • PHP has great unit testing tools

How Do I Get Started?

  • Use it to automate a boring, repetitive task
  • Write a test for another programmer
  • Don't worry about covering the entire codebase
  • Start with one test for your next feature/bugfix

Online Help - not so helpful

  • Major breaking syntax changes between v1 and v2
  • Headless browser testing requires v2
  • Most online examples are for v1
  • Selenium+PhantomJS tutorials are incorrect
  • Current PhantomJS (2.0) binary release is broken!

We only need these three

  • PHPUnit 4.8
  • Selenium Stand-Alone Server 2.47.0
  • PhantomJS 2.0 (Patched)

These two helper utilities are optional

  • Selenium IDE 2.9.0
  • Selenium 2 PHP Formatter                  

Required Prerequisites

(You probably already have them installed)

  • PHP 5.3.3 (or higher)
  • Web server w/ PHP Configured
  • Java Runtime
  • Firefox
  • Xdebug Extension (for code coverage)
